## Tech Stack Fijo
- **Base**: Fork de [Polymarket/agents](https://github.com/Polymarket/agents) (integra Gamma API + trade execution)
- **Lang**: Python 3.9+
- **API**: py-clob-client (en agents), WebSocket para real-time
- **Wallet**: web3.py + Polygon RPC (Amoy testnet para tests)
- **AI**: OpenAI API (opcional, via agents utils)
- **Testing**: pytest + historical data de Bitquery/FinFeed
- **Deps**: Ver requirements.txt (incluye pydantic, rich, chroma)
